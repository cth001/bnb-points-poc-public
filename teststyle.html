Token Transfer

TOK

API Documentation

POST https://api.yourservice.com/v1/transactions

Request Parameters

  • from string (required)
    Sender's wallet address
  • to string (required)
    Recipient's wallet address
  • value integer (required)
    Transfer amount in tokens

Example Request

{
  "from": "0x3F5...C4B",
  "to": "0x892...D1A",
  "value": 100
}
